# Smart Expense Tracker

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Platform](https://img.shields.io/badge/platform-web-lightgrey.svg)

> A modern, mobile-friendly expense tracker application to help you manage your finances smartly. Track income, expenses, set budgets, and get alerts when you're overspending.

🔗 **Live Demo:** [hashpieris.github.io/smart-expense-tracker](https://hashpieris.github.io/smart-expense-tracker/)

---

##  Features

| Feature | Description |
|---------|-------------|
|  **Income & Expense Tracking** | Record all your financial transactions with ease |
|  **Category Management** | 8 predefined categories (Food, Transportation, Shopping, Entertainment, Health, Bills, Income, Others) |
|  **Visual Statistics** | Interactive doughnut chart showing expense distribution by category |
|  **Smart Filtering** | Filter transactions by today, this week, this month, or all time |
|  **Budget Alerts** | Set monthly budget and receive warnings when exceeding 80% or 100% |
|  **Local Storage** | All data saved in your browser — no account needed, data persists after refresh |
|  **Export Data** | Export your transaction history as JSON file |
|  **Responsive Design** | Works perfectly on mobile, tablet, and desktop |

##  Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources)

### Installation

1. **Clone the repository**
  
   git clone https://github.com/HashPieris/smart-expense-tracker.git

2. **Navigate to project folder**

    cd smart-expense-tracker

3. **Open the application**

    Simply open index.html in your browser
    Or use a local server:


# Using Python
    python -m http.server 8000

# Using VS Code Live Server extension
 How to Use
Adding a Transaction
Click "Tambah Transaksi" button

Select transaction type (Income/Expense)

Choose category

Enter amount and description

Pick date

Click "Simpan"

Setting a Budget
Enter your monthly budget in the budget input field

Click "Set Budget"

Visual progress bar will show your spending status

Filtering Transactions
Click "Hari Ini" — see today's transactions

Click "Minggu Ini" — see this week's transactions

Click "Bulan Ini" — see this month's transactions

Click "Semua" — see all transactions

Exporting Data
Click "Export" button

Your data will be downloaded as a JSON file
